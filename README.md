# Work Management System

## About the project

Simple Java based application for tracking employee shifts and wages.

## Features

- Create and manage employee profiles

- Secure login with password verification

- Track shift hours with automatic calculation of hours, minutes, and seconds

- Calculate earned pay per shift and maintain unpaid salary

- Administrator privileges: edit employee info, grant admin rights

- SQLite database backend

## Installation

1. Clone the repository: git clone https://github.com/KavinRamasaamy/WorkManagementSystem.git

2. Open in IntelliJ IDEA or your preferred Java IDE.

3. Ensure you have JDK 17+ installed.

4. Run the Application.java class to start the program.

5. SQLite database will be created automatically in the data/ folder.

## Usage

On launch, you can: 

- Create a new employee profile

- Log in as an existing employee

- Admin users can edit employee data or record shifts for themselves.

- Regular employees can track their shifts and see earnings.
